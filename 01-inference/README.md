# Inference Recipes

Recipes and examples for text, vision, audio inference with vLLM-Omni's omni-modality support.

## Topics

This section will cover:

- Text generation with various sampling strategies
- Vision language model (VLM) inference
- Audio processing and generation
- Streaming inference for real-time responses
- Batch processing for efficiency
- Multi-modal input handling

## vLLM-Omni Inference Features

- **Multi-modal Inputs**: Process text, images, video, and audio
- **Flexible Architectures**: Support for both autoregressive and non-autoregressive models
- **Heterogeneous Outputs**: Generate text, images, or multimodal responses
